# 数据来源

- **猎聘招聘网**

## 爬取方式

通过浏览器抓包判断出岗位数据api，然后通过selenium模拟真人滑动验证条，然后提取网页接口数据，存储到csv文件



# 数据处理与分析

先确定网页图表信息再做处理，本项目清洗了薪资，城市名。

数据清洗后根据echarts图表所需数据进行数据处理
